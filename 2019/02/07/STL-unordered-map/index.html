STL-unordered_map

STL

基础定义

  • map是一个有序的映射(键值排序),unordered_map是一个无序的映射; 单独的头文件<unordered_map>
  • unordered_map以散列代替map内部的红黑树,速度比map快得多;

函数

size_type count(key):如果unordered_map中没有该键值,那么返回0;(用于检验key是否存在)
unordered_map::iterator it; :通过定义迭代器访问元素, it->first访问key it->second访问value
