常见数据类型的转换

Practice

int转换string

字符串流操作

  • 通过字符串流进行操作:istringstream,ostringstream,stringstream ; 需要头文件
  • 字符串流在进行多次转换时,每次读入数据不会将上次缓冲区的内容覆盖掉,而是增添到尾部;如果需要清除缓冲区,需要使用clear();
  • 在C++中,一般采用库进行数据转换,更加安全,可通过重载来支持各种类型的转换.

string转换char[]

通过str.copy()

size_t string::copy(char* s,size_t len,size_t pos=0)
pos:字符串的起始位置,默认为0;
len:选取字符串的长度;
返回值:实际选取字符串长度,一般和参数len相等,除非pos+len超过字符串的长度;
特点:不会为数组添加’\0’,需要手动添加’\0’;

通过strcpy()

const char* c_str() const noexcept;
将字符串转换成为字符数组,在数组后面添加’\0’;

char strcpy(char destination,const char* source)
strcpy是C Library的函数,操作数为两个char[];
特点:source的所有内容(从开始到终止符’\0’,包括终止符)传给destination;
